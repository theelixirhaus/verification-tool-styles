Typography

All icons used on the style guide are provided from FontAwesome.. You can output these icons anywhere by using a base set of code, and changing the icons by making one slight modification.

If you want to output the icon of a camera, you simply use the following HTML:

<i class="fa fa-camera" aria-hidden="true"></i>

In this example, you could change the word "camera" (but don't lose the "fa-") to any of the other FontIcon names. You can view the full list of FontAwesome icons as well as their corresponding codes.
