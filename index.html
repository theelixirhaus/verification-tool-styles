Iconography

All icons used on the style guide are provided from Ionicons. You can output these icons anywhere by using a base set of code, and changing the icons by making one slight modification.

If you want to output the icon of a camera, you simply use the followieng HTML:

<i class="ion-ios-camera" aria-hidden="true"></i>

You can view the full list of Ionicons icons as well as their corresponding codes.
